- Login
- Validação de Login
- Buscar Consultas
- Realizar novo agendamento de consulta

# Rodando o Projeto


```
# Clone esse repositorio
git@github.com:washingtonbsb/desafio-tecnico-frontend-conexa.git

# Entre no repositorio
cd backend

# Instale as dependencias
npm install

# Inicie o back
npm run start

# Entre no repositorio
cd frontend

# Instale as dependencias
npm install

# Inicie o front
npm run start


```
